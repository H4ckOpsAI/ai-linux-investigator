* Format findings
* Generate remediation recommendations

## Risk Scoring System

Implemented severity-based scoring.

| Severity | Score |
| -------- | ----- |
| HIGH     | 30    |
| MEDIUM   | 15    |
| LOW      | 5     |

Example:

```text
Suspicious Process       HIGH
Network Listener         HIGH
Persistence Mechanism    HIGH
Login Activity           LOW
```

Score:

```text
30 + 30 + 30 + 5 = 95
```

Maximum score:

```text
100
```

---

## Risk Classification

Implemented overall risk classification.

```text
0 - 39     = LOW
40 - 69    = MEDIUM
70 - 100   = HIGH
```

Current investigation result:

```text
Risk Score: 95/100
Risk Level: HIGH
```

## Login Analyzer Improvements

Issue:

Duplicate login events were appearing inside reports.

Example:

```text
User testuser logged in from 172.18.0.2
User testuser logged in from 172.18.0.2
```

Resolution:

Implemented duplicate detection using:

```python
seen = set()
```

Result:

Only unique login events are included in findings.
